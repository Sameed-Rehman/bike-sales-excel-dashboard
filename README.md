# Bike Sales Data Cleaning and Dashboard (Excel Project)

## Project Overview
This project demonstrates a complete Excel-based analytics workflow using bike sales data. The work includes data cleaning, feature engineering (bucketing), pivot analysis, interactive visualizations, and a dashboard with slicers to support decision-making.

The dashboard helps analyze bike purchase trends by:
- Gender
- Commute Distance
- Age Brackets
- Education
- Marital Status

## Files Included
- `Raw Dataset.xlsx`  
  Original dataset before cleaning and transformation.
- `Excel Project Cleaning and Dashboard.xlsx`  
  Final project workbook containing cleaned data, derived buckets, pivot tables, charts, and the interactive dashboard.

## What Was Done
1. Data Cleaning
- Checked for inconsistencies and missing values.
- Standardized fields where required.
- Prepared the dataset for reliable analysis.

2. Data Transformation
- Created useful buckets/derived fields (for example: age brackets).
- Structured data for pivot-based analysis.

3. Analysis
- Built pivot tables to compare bike sales across key demographics and behavior factors.
- Evaluated sales patterns by gender, commute distance, and age groups.

4. Dashboard Development
- Created charts and visual summaries for quick insight.
- Added slicers for interactive filtering.
- Enabled dashboard updates by dimensions such as education and marital status.

## Dashboard Usage
Open `Excel Project Cleaning and Dashboard.xlsx`, then use slicers to filter and explore trends across segments.

## Key Business Questions Answered
- Which customer segments are more likely to purchase bikes?
- How does commute distance relate to bike purchases?
- How do age and gender influence buying behavior?
- How do education and marital status affect sales trends?
